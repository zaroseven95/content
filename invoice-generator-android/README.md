# Invoice Generator (Android)

A simple Android app for phone users to create invoices quickly, calculate totals with tax, and share the invoice text through any messaging or email app.

## Features
- Add customer name
- Add line items (description, quantity, unit price)
- Automatic subtotal, tax, and grand total calculation
- Share invoice text using Android's share sheet

## Build
1. Open this folder in Android Studio Hedgehog+.
2. Let Gradle sync.
3. Run on a device/emulator (API 24+).
